# Xbox_360_Playnite_Theme
Custom Playnite theme and extension to mimic the vibe of the Xbox 360.

Installable through the Playnite Addons Browser. **Make sure to install both the theme and the extension, as both are required.**

Keep an eye out for updates as I continue to work on this. Updates can be automatically installed through the Playnite Addons Browser as well.
